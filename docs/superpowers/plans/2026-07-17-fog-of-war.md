# Fog-of-war (stanze visitate) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Marcare le stanze visitate nel runtime graph e disegnare sulla mappa le non visitate come nodi scuri anonimi (`???`), rivelando la topologia ma non il contenuto.

**Architecture:** Nuova datatype property `sr:visited` (Room→boolean) nel vocabolario runtime, scritta dal motore quando il player entra in una stanza; una shape runtime di forma; `build_map` nasconde nome e contenuto delle stanze non visitate quando c'è uno stato di partita.

**Tech Stack:** Python 3.12 (venv `.venv/`), rdflib, pySHACL, pytest. Nessuna dipendenza nuova.

**Spec:** `docs/superpowers/specs/2026-07-17-fog-of-war-design.md`

## Global Constraints

- Eseguire tutto con `.venv/Scripts/python` (Windows, shell Git Bash → forward slash).
- Namespace: `sr:` = vocabolario, `ex:` = istanze. Annotazioni bilingui `@it`/`@en` obbligatorie su ogni risorsa nuova (property e shape); le shape hanno anche `sh:name`/`sh:description` bilingui.
- Convenzione runtime: **assenza di tripla = stato di default**; `sr:visited true` si scrive solo quando la stanza è visitata (mai `false`).
- Il fog vale **solo con uno stato di partita**: `build_map` senza player nello stato resta identica a oggi (mondo authored pieno).
- Su Windows lo stdout dei subprocess è cp1252: il tool `tools/map.py` fa già `sys.stdout.reconfigure(encoding="utf-8")`; i test leggono i file con `encoding="utf-8"`.
- I test esistenti (75) restano verdi a ogni task.

Sostituire `_node_label` per nascondere le non visitate quando c'è una partita in corso:

```python
def _node_label(world: Graph, state: Graph, room: URIRef,
                player_room: URIRef | None) -> str:
    """Etichetta multiriga del nodo stanza: nome, mostri, NPC, obiettivi di quest.

    In una partita in corso (player presente) le stanze non ancora visitate
    restano anonime (\"???\"): topologia visibile, contenuto nascosto.
    """
    in_run = player_room is not None
    if in_run and (room, SR.visited, Literal(True)) not in state:
        return "???"
    parts = [label_it(world, room)]
    if room == player_room:
        parts[0] = "🧝 " + parts[0]
    for monster in sorted(world.objects(room, SR.hostsMonster), key=str):
        icon = "👑" if (monster, RDF.type, SR.Boss) in world else "👹"
        if (monster, SR.isAlive, Literal(False)) in state:
            icon = "☠"
        parts.append(f"{icon} {label_it(world, monster)}")
    for npc in sorted(world.subjects(SR.npcInRoom, room), key=str):
        parts.append(f"🧙 {label_it(world, npc)}")
    for quest in sorted(state.subjects(SR.questStatus, Literal("active")), key=str):
        if (quest, SR.questTargetRoom, room) in world:
            parts.append(f"⭐ {label_it(world, quest)}")
    return "<br/>".join(parts)
```
